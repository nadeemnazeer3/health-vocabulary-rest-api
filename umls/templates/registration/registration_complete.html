Thank you for Registering

Please note down your access paramaters.

user: {{token_data.user}}
token: {{token_data.token}}

Now you can access any API by sending a request to it, and including user and token as request parameters (either GET or POST). Accessing the api(s) using curl might look like: 

curl -d "user={{token_data.user}}&token={{token_data.token}}" http://23.239.3.60/concepts?term=cancer


or in browser: 
    http://23.239.3.60/concepts?term=cancer&user={{token_data.user}}&token={{token_data.token}}
